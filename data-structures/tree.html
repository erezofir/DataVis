Binary Tree

What is it?

A Binary Tree is a hierarchical data structure where each node has at most two children: a left child and a right child.

Time and Space Complexity

Operation Binary Tree Binary Search Tree
Search O(n) O(log n)
Insert O(n) O(log n)
Delete O(n) O(log n)

Properties

  • The topmost node is called the root.
  • Each node can have at most two children.
  • Binary Trees can be balanced or unbalanced.
  • They are used in searching, sorting, and hierarchical relationships.

Advantages and Disadvantages

✅ Represents hierarchical data effectively.

✅ Efficient searching in balanced BSTs.

✅ Supports dynamic resizing.

❌ Can become unbalanced (reducing efficiency).

❌ Traversal can be time-consuming.

❌ Complex implementation for balancing.
