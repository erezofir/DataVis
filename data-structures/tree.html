Tree

What is it?

A Tree is a hierarchical data structure consisting of nodes, where each node has a parent (except the root) and zero or more children.

Time and Space Complexity

Operation Binary Tree Binary Search Tree
Search O(n) O(log n)
Insert O(n) O(log n)
Delete O(n) O(log n)

Properties

  • Root node is the topmost node.
  • Each node has zero or more children.
  • Can be used to implement various structures (e.g., heaps, BSTs).

Advantages and Disadvantages

✅ Great for hierarchical relationships.

✅ Supports efficient searching in BSTs.

✅ Dynamic resizing.

❌ Can become unbalanced.

❌ Traversal can be time-consuming.

❌ Complex to implement certain operations.
