Graph

What is it?

A Graph is a collection of nodes (or vertices) connected by edges. It can be directed or undirected, weighted or unweighted.

Time and Space Complexity

Operation Adjacency List Adjacency Matrix
Add Vertex O(1) O(V²)
Add Edge O(1) O(1)
Search O(V + E) O(V²)

Note: V represents the number of vertices, and E represents the number of edges in the graph.

Properties

  • Vertices (nodes) and edges (connections).
  • Can be directed or undirected.
  • Useful in modeling real-world relationships (e.g., social networks).

Visualization

Enter edges in the format: [[1,2],[2,3],[3,1]]

Advantages and Disadvantages

✅ Flexible and versatile representation.

✅ Models real-world systems effectively.

✅ Efficient for sparse data structures with adjacency lists.

❌ Memory overhead for adjacency matrix.

❌ Traversal can be computationally expensive.

❌ Implementation complexity.
