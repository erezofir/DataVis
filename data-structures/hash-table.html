Hash Table

What is it?

A Hash Table is a data structure that stores data in key-value pairs. It uses a hash function to compute the index of the data in an array.

Time and Space Complexity

Operation Time Complexity
Insert O(1)
Search O(1)
Delete O(1)

Properties

  • Key-value pair storage.
  • Fast lookup via hash function.
  • Collision handling via chaining or open addressing.

Operations

  • Insert: Add a key-value pair.
  • Search: Find a value by its key.
  • Delete: Remove a key-value pair.
