Array

What is it?

An array is a collection of items stored at contiguous memory locations. It allows random access and manipulation of data elements.

Time and Space Complexity

Operation Time Complexity
Access O(1)
Search O(n)
Insert O(n)
Delete O(n)

Properties

  • Contiguous memory allocation.
  • Fixed size (in static arrays).
  • Efficient access via index.
  • Homogeneous data elements.
  • Can be single-dimensional or multi-dimensional.

Operations

  • Access by index.
  • Insert and delete at specific positions.
  • Traversal to process all elements.

Advantages and Disadvantages

✅ Efficient random access with index.

✅ Memory-efficient for static arrays.

✅ Simple implementation and traversal.

❌ Fixed size for static arrays.

❌ Expensive insertions/deletions in dynamic arrays.

❌ Wasted memory in sparse usage.
