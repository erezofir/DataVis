Stack

What is it?

A Stack is a Last-In-First-Out (LIFO) data structure. Operations include Push, Pop, and Peek.

Time Complexity

OperationTime Complexity
PushO(1)
PopO(1)
PeekO(1)

Features

  • Follows LIFO principle
  • Efficient operations
  • Can be implemented using arrays or linked lists

Actions

  • Push: Add an element to the top
  • Pop: Remove the top element
  • Peek: View the top element without removing it
  • Clear All: Reset the stack

Advantages and Disadvantages

Advantages

  • Simple and efficient
  • Easy to implement

Disadvantages

  • Limited by stack size
  • No random access to elements

Possible Implementations

Doubly Linked List

A linked list with nodes connected in both directions.

Array

A fixed-size or dynamic array to store elements.
