Queue

What is it?

A Queue is a First-In-First-Out (FIFO) data structure. Elements are added at the rear and removed from the front.

Time and Space Complexity

Operation Time Complexity
Enqueue O(1)
Dequeue O(1)
Peek O(1)

Properties

  • Follows FIFO principle.
  • Efficient operations at both ends.
  • Can be implemented using arrays or linked lists.

Operations

  • Enqueue: Add an element to the rear.
  • Dequeue: Remove an element from the front.
  • Peek: View the front element without removing it.

Advantages and Disadvantages

✅ Simple to implement.

✅ Efficient for certain types of processing tasks.

✅ Useful in scheduling systems.

❌ Fixed size in static arrays.

❌ Inefficient when random access is needed.

❌ Operations depend on the implementation method.
