Linked List

What is it?

A Linked List is a linear data structure where each element is a node. Each node contains a value and a reference to the next node in the sequence. Linked Lists allow for dynamic memory allocation and efficient insertion or deletion operations.

Time and Space Complexity

Operation Time Complexity
Insertion O(1)
Deletion O(1)
Traversal O(n)

Properties

  • Dynamic size: Can grow or shrink as needed.
  • Efficient insertions and deletions.
  • Memory overhead due to storing references/pointers.
  • Linear traversal time for search operations.

Operations

  • Insert at Head: Adds a new node to the beginning of the list.
  • Delete by Value: Removes a node containing a specified value.
  • Traverse: Iterates through the linked list to access each element.

Pros and Cons

✅ Dynamic size, flexible memory usage.

✅ Easy to implement efficient insertion and deletion.

✅ Supports various types (singly, doubly, circular).

❌ Linear search time for finding elements.

❌ Extra memory overhead for pointers.

❌ Complex implementation for certain operations.
